VPN Configuration

Pelicula routes all download traffic through ProtonVPN. You need a paid plan (Plus or higher) with P2P and port forwarding support.

Go to protonvpn.com/account → WireGuard → Create key for Linux.
Do not enable "Moderate NAT" — it breaks port forwarding.
Copy the PrivateKey value from the generated config.

P2P-friendly countries: Netherlands, Switzerland, Sweden, Iceland, Romania

Remote Access Peligrosa

Expose Jellyfin to the internet via a hardened nginx vhost. Requires a public DNS hostname and open router ports. The admin dashboard stays LAN-only.

Public DNS hostname for this server (e.g. home.duckdns.org). Required for Let's Encrypt — must not be a raw IP.

Used for expiry notices and account recovery.

Staging issues untrusted test certs with no rate limits. Enable while testing, disable for production.

Drop fullchain.pem and privkey.pem into <CONFIG_DIR>/certs/remote/ before restarting.

Self-signed certificates are rejected by iOS/tvOS Jellyfin apps and most browsers without a manual trust override.
