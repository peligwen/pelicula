ProtonVPN Setup

Required for downloading media. You can skip this and set it up later.

Go to protonvpn.com/account → WireGuard → Create key for Linux.
Do not enable "Moderate NAT" — it breaks port forwarding.
Requires a paid plan (Plus or higher) with P2P and port forwarding.

Server: Netherlands (best for P2P availability)
Why ProtonVPN? Do you work for them?
Nope. Pelicula uses Gluetun for VPN, which supports 30+ providers. We default to ProtonVPN because it's the only provider that reliably offers WireGuard + port forwarding + P2P on all paid plans — the three things a download client needs. You can switch providers later by editing .env and the Gluetun config.
